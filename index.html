Apresentação

Bem-vindo à página do curso de programação Python. Neste curso, elaborado em parceria com o Encontro Acadêmico de Modelagem Computacional do LNCC (EAMC 2019), você terá contato com a linguagem Python. Este material é mais apropriado para estudantes com alguma familiaridade com conceitos básicos de programação, que desejem conhecer aspectos da linguagem Python. Na primeira metade do curso veremos aspectos básicos da linguagem, incluindo estruturas de dados, estruturas de controle (loops e condicionais), funções e classes. Em seguida, exploraremos algumas das bibliotecas mais usadas no contexto da computação científica e ciência de dados.

Ambiente

Opção 1: Instalação local

Esta opção é mais avançada, recomendada para os que desejarem configurar o ambiente de programação em sua própria máquina. Para tal, você deverá:

  1. Instalar a distribuição Anaconda, observando os requisitos para seu sistema operacional. A instalação é bem simples: basta fazer o download do instalador e seguir os passos recomendados;
  2. Clonar (ou fazer o download) do repositório do curso, no diretório de sua preferência;
  3. Instalar os pacotes listados no arquivo sci_env.yml, contido no repositório. Você também pode criar um ambiente virtual usando o arquivo. Para isso, execute o código conda env create --file sci_env.yml, o que também exige que você ative o ambiente executando source activate sci_env (Linux) ou activate sci_env no Anaconda Prompt (Windows);
  4. No diretório contendo o material do curso, você deverá então inicializar o ambiente Jupyter Notebook executando jupyter notebook no Anaconda Prompt (Windows) ou no próprio terminal (Linux). O ambiente Jupyter Notebook executará no próprio navegador, no endereço localhost:8888. Você poderá então navegar, no próprio browser, para o diretório `notebooks`, contendo os materiais das aulas.
Opção 2: Execução em nuvem

Esta é a opção mais simples. Usaremos o ambiente Google Colab, que executa código como serviço em nuvem e, portanto, não exige configuração local. É necessário apenas ter uma conta na Google. Os links para os notebooks estão na aba `Conteúdo` desta página.

Aulas

Aula 1. Estruturas de dados fundamentais

São apresentadas os principais tipos e estruturas de dados nativas da linguagem Python.

(Abrir no Google Colab / Visualizar html)

Aula 2. Estruturas de controle, Funções e Classes

Nesta aula aprenderemos a alterar o fluxo de execução de nossos programas usando estruturas de controle (condicionais e de repetição). Aprenderemos também como encapsular códigos em funções. Finalmente, veremos como o conceito de classes permite ao programador trabalhar com o paradigma de programação orientada a objetos.

(Abrir no Google Colab / Visualizar html)

Aula 3. Python para Computação Numérica - Pate 1: NumPy

Aqui veremos uma introdução às funcionalidades básicas de um dos princípais pacotes para computação científica em Python, a NumPy. Sua principal característica é a capacidade lidar eficientemente com arrays multidimensionais.

(Abrir no Google Colab / Visualizar html)

Aula 4. Python para Computação Numérica - Parte 2: SciPy

A biblioteca SciPy oferece um conjunto de pacotes para a resolução de problemas científicos. Veremos algumas de suas principais aplicações, como por exemplo a solução de sistemas (lineares e não-lineares), otimização não-linear e Equações Diferenciais Ordinárias.
